# Bug Bounty Vault

🛡️ Plataforma educativa de inteligencia viva sobre **Bug Bounty**.

Este repositorio está diseñado como una base privada/personal de estudio estratégico para construir criterio, método y enfoque profesional en bug bounty. La idea es centralizar en un solo lugar recursos de alto valor: extensiones, PDFs, cursos, hunters élite, writeups, metodología y una guía clara para llegar a los primeros **$500–$1000 USD**.

---

## Estructura

- `README.md` → visión general del vault
- `roadmap/` → rutas de aprendizaje y plan de ataque
- `resources/` → extensiones, herramientas, PDFs, cursos y laboratorios
- `intel/` → hunters destacados, plataformas y referencias de alto nivel
- `guides/` → guías accionables paso a paso
- `notes/` → espacio para notas, writeups personales y observaciones

---

## Objetivo principal

Construir una base sólida para:

- Aprender bug bounty con metodología real.
- Organizar recursos de forma útil y estratégica.
- Documentar patrones de vulnerabilidades repetibles.
- Diseñar un sistema realista para conseguir los primeros payouts.
- Evolucionar de estudiante a hunter con criterio profesional.

---

## Enfoque

Este vault está orientado a:

- Web hacking
- API security
- Reconnaissance
- XSS, IDOR, Access Control, SSRF, Auth flaws
- Writeups y triage thinking
- Productividad y sistematización del hunting

---

## Meta inicial

**Primeros $500–$1000 USD en Bug Bounty** mediante:

1. Recon efectivo
2. Hunting en programas adecuados para principiantes
3. Repetición de vulnerabilidades probables
4. Reportes claros y profesionales
5. Documentación de hallazgos y aprendizaje continuo
